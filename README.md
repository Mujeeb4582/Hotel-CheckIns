# ğŸ“– [Hotel CheckIns] <a name="about-project"></a>

> The app allows users to add and view details about a person's check-ins at a hotel. The main page displays a table of existing check-ins, with the ability to add new check-ins through a modal form. Clicking on a row in the table displays the details of that check-in in a right-side drawer. This project uses the Ant Design library for its UI components and is written in TypeScript.

### Prerequisites

In order to run this project you need:
- [Node](https://nodejs.org/en/) installed in your computer
- [Git](https://git-scm.com/) installed in your computer

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:Mujeeb4582/Hotel-CheckIns.git
```

### Install

Install this project with:

```sh
  cd Hotel-CheckIns
  npm install
```
### Usage

To run the project, execute the following command:
